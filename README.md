# Xbot プロジェクト

## プロジェクト概要
このプロジェクトは、X（旧Twitter）への自動投稿を生成AIが自動的にタスクをこなして対応するシステムです。AIを活用して、定期的な投稿の自動化や、特定の条件に基づいた投稿の生成・実行を行います。YouTubeのトレンド分析と外国人視点のリサーチを組み合わせ、その日のバズりトピックを元に、外国人が共感・笑えるようなニッチで親しみやすい投稿を生成します。
